# react-native-pdf-text-extractor

Extract contents from pdf files

## Installation

```sh
npm install react-native-pdf-text-extractor
```

## Usage


```js
import { multiply } from 'react-native-pdf-text-extractor';

// ...

const result = await multiply(3, 7);
```
